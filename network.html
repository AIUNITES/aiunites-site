The Middleware Stack

What every site shares — the infrastructure that makes them a network, not a portfolio.

🔐

Cross-Site Identity

Shared user table in SQLite. Register once, log in anywhere. Session-based auth with demo mode for try-before-you-signup. First user auto-becomes admin.

☁️

CloudDB Sync

Every site has cloud-database.js deployed. Local data syncs to cloud storage via Google Forms + Apps Script. Works offline, syncs when connected. Deployed to all 18 sites.

🔗

Webring Navigation

Fixed top bar on every site. Highlights the current site. Auto-converts URLs for local development. One navigation context across the entire network.

📝

Domain Protocols

MNN (Muscular Neuro Notation) for human movement across BodSpas and InThisWorld. VRN (Voice Resonance Notation) for vocal training on VoiceStry. Open text formats, machine-parseable.

🎨

DemoTemplate

Base template for all sites. Dark theme, CSS variables, vanilla JS. Auth system, admin panel, settings modal, backup/restore, cache viewer, legal modals. New sites clone this and customize.
