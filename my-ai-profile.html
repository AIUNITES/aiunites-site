How It Works

Three steps. Works with any AI tool that accepts a system prompt or first message.

1

Fill in your profile

Use the builder to answer a few questions — your role, what you're working on, how you prefer to communicate, your technical constraints. Takes about 5 minutes.

2

Download your JSON file

Your profile is saved as a clean JSON file — my-ai-profile.json — that you keep on your machine. Update it anytime as your projects change.

3

Paste it into any AI

Start your next Claude or ChatGPT session with: "Here is my context: [paste JSON]" — or upload the file as an attachment. The AI immediately knows who you are.

4

Update the session field

One field in your profile is session.focus — swap it out each time to tell the AI what you're working on right now. Everything else stays the same.

Frequently Asked Questions

What is a portable AI profile?

A portable AI profile is a JSON file that describes who you are — your role, expertise level, current projects, communication preferences, and hard constraints. Instead of repeating this context every time you start an AI session, you paste the file once and the AI immediately understands who it’s talking to. Unlike built-in AI memory, it works across any tool: Claude, ChatGPT, Gemini, Perplexity, or any LLM that accepts a system prompt.

How is this different from Claude Memory or ChatGPT Memory?

Built-in memory is locked to one tool. If you use Claude for code and ChatGPT for writing, you have to set up context separately in both. Your AIUNITES AI Profile is a file you own — it travels with you to any AI tool, any device, and any platform. You control it, update it, and it never gets out of sync between tools.

Is my data stored on your servers?

No. Your AI profile is stored entirely in your browser’s localStorage — it never leaves your device unless you explicitly download or paste it somewhere. AIUNITES has no server, no database, and no account required to use the builder. This is by design: your context is yours.

How do I give Claude context about me using this?

Build your profile in the AIUNITES builder, copy the JSON, then start your Claude session with: “Here is my context: [paste your JSON]”. Claude will read your role, projects, preferences, and constraints immediately. You can also upload the downloaded .json file as an attachment in Claude.ai. Update the session.focus field each time — everything else stays the same.

Can I have different profiles for different contexts?

Yes. The builder lets you save multiple named profiles in your browser — for example, a “Work” profile with your job context, a “Side Project” profile for personal builds, and a “Writing” profile with different tone and style preferences. Switch between them instantly and each generates a different JSON file tailored to that context.
